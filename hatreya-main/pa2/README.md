Hansa Atreya (hatreya)

pa2 includes the files:

Graph.h - Header file for Graph ADT, defining Graph type and its operations.

Graph.c - Implementation of Graph ADT. Uses BFS and adjacency lists.

GraphTest.c - Test client for Graph ADT, to verify function.

FindPath.c - Main client. Reads input and uses BFS to process vertices and their shortest paths.

List.h - Header file for the list ADT from pa1.

List.c - pa1 list ADT implementation. Used by Graphs.c

Makefile - Compiles code and checks for errors.

README.md - This file. Descries the content of pa2.
