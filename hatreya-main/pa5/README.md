# Hansa Atreya (hatreya)

- WordFrequency.c Implements a program that reads an input text file, extracts words, and counts their frequencies using the Dictionary ADT. Words are case-insensitive and consist of alphabetic characters only.
- Dictionary.c Implementation of a Dictionary Abstract Data Type using a hash table. Supports insertion, deletion (with tombstones), lookup, resizing, copying, and diagnostics.
- Dictionary.h Public interface for the Dictionary ADT. Defines data types and function prototypes used by client programs.
- DictionaryTest.c Test client used to verify Dictionary functionality including insertion, deletion, resizing, and equality checking.
- Makefile Provided starter Makefile.
